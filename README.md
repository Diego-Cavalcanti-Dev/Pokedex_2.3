# 📱 Pokédex

Aplicativo mobile desenvolvido em **Flutter** como projeto da disciplina de **Programação Mobile**. A aplicação consome a **PokéAPI** para exibir informações sobre Pokémon, permitindo visualizar detalhes e gerenciar uma lista de favoritos.

## 📖 Sobre o Projeto

A Pokédex tem como objetivo proporcionar uma experiência simples e intuitiva para consulta de Pokémon, apresentando informações obtidas em tempo real por meio de uma API REST pública.

O projeto foi desenvolvido seguindo boas práticas de organização de código e arquitetura, utilizando recursos de armazenamento local para persistência dos Pokémon favoritos.

---

## ✨ Funcionalidades

- 📋 Listagem de Pokémon
- 🔍 Pesquisa de Pokémon
- 📄 Visualização de detalhes
- ❤️ Adicionar e remover Pokémon dos favoritos
- 💾 Persistência dos favoritos utilizando armazenamento local
- 🌐 Consumo da PokéAPI
- 📱 Interface responsiva em Flutter

## 🚀 Como Executar

### Clone o repositório

```bash
git clone https://github.com/Diego-Cavalcanti-Dev/Pokedex_2.3.git
```

### Entre na pasta

```bash
cd Pokedex_2.3
```

### Instale as dependências

```bash
flutter pub get
```

### Execute o projeto

```bash
flutter run
```
